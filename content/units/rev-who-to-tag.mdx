  ### Collection: Working with AI Tools
---

For copy, visuals and docs, a relevant teammate reviewing the preview is enough. For anything touching authentication, user data, payments, or server logic, an engineer has to review the code — the system routes these automatically to the right people (that's what CODEOWNERS does). If in doubt, tag an engineer; it's never wrong.
