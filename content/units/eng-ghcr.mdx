GitHub Packages is GHCR. We build the image in GitHub Actions, scan it, push it to `ghcr.io`, and Coolify deploys by pulling that prebuilt image. Two reasons: you scan the exact artefact you ship (rebuilding on the server would ship something you didn't scan), and you keep heavy build load off the VPS (builds can and do crash small boxes). One-time setup gotcha: the server needs `docker login ghcr.io` with a PAT to pull private images, or the pull fails with `unauthorized`.
